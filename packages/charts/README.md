# React Financial Charts

> **Note:** This is an actively maintained fork of [react-financial-charts](https://github.com/react-financial/react-financial-charts) with **React 18 & 19 support**, new interactive tools, and **published on npm** under `@sgonzaloc` scope.

[![npm version](https://img.shields.io/npm/v/@sgonzaloc/react-financial-charts.svg)](https://www.npmjs.com/package/@sgonzaloc/react-financial-charts)

---

## Installation

```bash
npm install @sgonzaloc/react-financial-charts
```

All packages (`core`, `axes`, `scales`, `series`, `utils`, `interactive`, `indicators`) are included as dependencies.

---

## Features

-   ✅ **React 18 & 19 Ready**
-   ✅ **TypeScript** - Fully typed
-   ✅ **Storybook 8** - Interactive documentation
-   ✅ **New Interactive Tools** - Rectangle, Arrow, Price Range, Freehand Brush, Marquee Zoom
-   ✅ **Selection Support** - Click to select any interactive object
-   ✅ **Performance Optimizations** - `shouldComponentUpdate`

### Chart Types

-   Scatter, Area, Line
-   Candlestick, OHLC
-   HeikenAshi, Renko, Kagi, Point & Figure

### Technical Indicators

-   EMA, SMA, WMA, TMA
-   Bollinger Bands, SAR
-   MACD, RSI, ATR
-   Stochastic (fast, slow, full)
-   ForceIndex, ElderRay, Elder Impulse

### Interactive Drawing Tools

-   Trendline, Fibonacci Retracements
-   Gann Fan, Channel
-   Rectangle, Arrow
-   Price Range, Marquee Zoom
-   **Freehand Brush**
